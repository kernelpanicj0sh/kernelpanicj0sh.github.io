ncrypt.me(conceptionalised)

ncrypt.me is a project with a goal of providing an open vpn network that anyone can host through

proof-of-security (POS) and sand-to-the-box (STTB). The concept is that users can install

the ncrypt.me client and be given an option to connect to any vpn host or closet to there

location using simple ping requests. hosts will be limited with there control over the vpn

to avoid poisoned or hosts with malicous intent from being able to do things like monitor traffic

or user activity in-depth. hosts require an incentive for sharing there bandwidth, charging users

a monthly or annual subscription and giving a certain percentage to the host-pool is a good idea.
